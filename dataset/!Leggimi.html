File ad uso pubblico


(INCSTRAD) Rilevazione degli incidenti stradali con lesioni a persone
Anno 2017
L'archivio compresso (.zip) contiene, oltre questa guida introduttiva, i seguenti file, distinti nelle directory MICRODATI e METADATI:
file di microdati, tracciato record con classificazioni associate, documenti metodologici.
Il nome di ciascun file è caratterizzato dall'acronimo dell'indagine, dal contenuto dello stesso e dal periodo di riferimento dei dati. Il file di microdati, in formato txt, è fornito con campi separati da tabulatore e nella prima riga riporta il nome dei campi. Ogni campo delimitato corrisponde ad una variabile del tracciato record.
Le classificazioni sono raggruppate nella cartella METADATI/Classificazioni. Il nome del file di ciascuna classificazione è contrassegnato dal suffisso var seguito dal numero d'ordine presente nel tracciato record relativo alle variabili di tipo categorico.
Si ricorda che a causa del trattamento dei dati per la tutela della riservatezza, le elaborazioni effettuate sui file ad uso pubblico possono condurre a risultati in qualche misura difformi rispetto a quelli pubblicati.
