Guida alla conversione di file di testo (formato TXT) in file Excel

 

 

 

Avviare Excel / selezionare il menu File / quindi cliccare sulla voce Apri - Si aprirà questa finestra di ricerca:

Da qui si potrà procedere alla ricerca del file che è stato preventivamente estratto dall'archivio compresso.

Si ricorda che il file da ricercare ha estensione txt

Cliccare sul pulsante Apri. Verrà mostrata la sequente schermata che consente la importazione guidata :

Cliccare sul tasto Avanti  e poi scegliere il tipo di delimitatore dei campi (per default Tabulazione)

Se è necessario, è possibile impostare il formato delle colonne selezionandole una per volta ed eventualmente il separatore dei decimali

Dopo avere cliccato su Fine avremo il riempimento del foglio Excel (max righe per versione 2003=65,536 rows by 256 columns; per versione 2010=1,048,576 rows by 16,384 columns) 

 

Per poter salvare il documento in formato Excel, selezionare dal menu File, la voce Salva con nome

In questa finestra, selezionare il tipo di file "Cartella di lavoro di Excel" e quindi cliccare su Salva

Il file ottenuto sarà un file convertito da formato testo ad Excel
